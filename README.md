# pokedexcli

Use the Makefiles after you clone the repo to try the project !
`make help` for more infos;

This project is my first Go project. It is a small REPL-based Pokédex CLI that interacts with the [public PokeAPI](https://pokeapi.co/docs/v2) to explore the Pokémon world from the terminal.

Through this project, i practiced several core Go concepts such as structs, maps, pointers, package organization, HTTP requests, JSON decoding, simple caching with goroutines and mutexes, and command handling in a REPL loop.
It also helped me better understand how to structure a small CLI application in Go and how to separate responsibilities across internal packages, trying to follow idiomatic go and industry conventions.

The project uses the public [PokeAPI v2](https://pokeapi.co/docs/v2) to fetch data about location areas and Pokémon, then let the user explore, catch, and inspect Pokémon directly from the command line.

Some things could be improve, features added, but it's alright for now.
